If any web page contains input fields than this extension can help to batch-fill all those fields from simple formatted text. Simply paste text into the text box below. Every line will be copied into input fields, successively, starting from the nominated "First input field" (defaults to 0, i.e. the first one found). After opening this extension, all found input fields are highlighted with a red background, and if you hover the mouse over them, you see there respective index. This can help you to identify the right "First input field" and enter it below.
